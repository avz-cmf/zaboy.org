Class Zaboy_Dic

Zaboy_Dic

Zaboy_Dic is dependency injection container class (Dic). Dic load as resurce plugin of Bootstrap Zaboy_Application_Resource_Dic.

Add in application.ini for load Zaboy_Dic

resources.dic[] =

You can change class of Zaboy_Dic for load:

resources.dic.class = My_Dic_Class

For more information about configuration see Zaboy_Dic_ServicesConfigs and Zaboy_Application_Resource_Dic
*

You can get Dic as resource:

$dic = $bootstrap->getResource('dic');

Objects and Services for DIC
Zaboy_Dic can load any object which corresponds a some requirements.
Zaboy_Dic also can contane that object if it is described in Services Config ( in most cases it is application.ini). In that case this object is Service.
Objects, which was loaded via Dic is not stored in Dic. Every call Zaboy_Dic::get() will return new object. For Services you will get singleton instance be default. You can change this behavior in Service config. See Zaboy_Dic_ServicesConfigs for details.

For more information about Objects, Services and requirements for them, see Zaboy_Services
