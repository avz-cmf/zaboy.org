Zaboy_Dic

Zaboy_Dic is dependency injection container class (Dic). Dic is loading as resurce plugin of Bootstrap { @see Zaboy_Application_Resource_Dic}.

Add in application.ini for load { @see Zaboy_Dic}

resources.dic[] =

You can change class of Zaboy_Dic for load:

resources.dic.class = My_Dic_Class

For more information about configuration see Zaboy_Dic_ServicesConfigs and Zaboy_Application_Resource_Dic
*

You can get Dic as resource:

$dic = $bootstrap->getResource('dic');

Objects and Services for DIC
Zaboy_Dic can load any object which corresponds a some requirements.
Zaboy_Dic also can contane that objectif it is described in Services config ( in most cases it is application.ini). In that case it is Service.
For more information about Objects, Services and requirements for them, see Zaboy_Service
