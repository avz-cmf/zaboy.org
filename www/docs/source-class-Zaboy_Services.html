 14: /**
 15:  * Base class for Services
 16:  * 
 17:  * <b>General remarks about Service Object and Service Class</b>
 18:  * 
 19:  * There are 5 different entities: Service, Service Name, Service Object, Service Class and Servise Config<br>
 20:  * 
 21:  * <br><b>Service</b><br>
 22:  * Service is PHP object (Service Object) which was loaded and registered in {@see Zaboy_Dic} via {@see Zaboy_Dic::get()} with Service Name<br>
 23:  * It is possibly only if Service is described in Service Config<br>
 24:  * 
 25:  * <br><b>Service Config</b><br>
 26:  * In most cases Service Config is a part of application.ini
 27:  * <pre> 
 28:  * resources.dic.myService.class = My_Service_Class
 29:  * ; My_Service_Class is Service Class 
 30:  * </pre>
 31:  * Mor about rules of desrcribe Services in Service Config see {@link Zaboy_Dic_ServicesConfigs}
 32:  * 
 33:  * <br><b>Service Class</b><br>
 34:  * There isn't any special interface for Service Classes.
 35:  * Class of Service Object may be inherited from {@see Zaboy_Services} or not.
 36:  * There are some requirements for (@see Zaboy_Services::__construct()} definition.<br>
 37:  * Next variants is possibly:
 38:  * <code> 
 39:  * //constructor is allowed not to have parameters
 40:  * function __construct()
 41:  * 
 42:  * //there is $options only. Param has to have name $options
 43:  * function __construct(array $options)
 44:  * 
 45:  * //There are some object type parameters with specified class name  
 46:  * function __construct(Class_Name $object1, Another_Class $object2, ...)
 47:  * //or 
 48:  * function __construct(array $options, Class_Name $object1,  Another_Class ...)
 49:  * //the $options if it exist as parametr for __construct have to be a first parameter.
 50:  * 
 51:  * //There are some parameters which is Services with specified class name or not
 52:  * function __construct(Service_Class $service1, Another_Service_Class $service2, ...)
 53:  * function __construct(array $options, Service_Class $service1, Another_Service_Class ...)
 54:  * 
 55:  * //According to requirements for Service describe, class of Service have to 
 56:  * //specified in Service Config. Because this reason, you may have not to notify classes
 57:  * // for Services in parameters, but it is recommended for type control. 
 58:  * function __construct(Service_Class $service1, $service2  ...)
 59:  * function __construct(array $options, $service1, $service2  ...)
 60:  * 
 61:  * //And all together:
 62:  * function __construct(array $options, Service_Class $service1, $service2, Class_Name $object1)
 63:  * //In this case we must have in Services config:
 64:  * </code>
 65:  * <pre> 
 66:  * resources.dic.service1.class = Service_Class
 67:  * resources.dic.service2.class = Next_Service_Class
 68:  * </pre>
 69:  * 
 70:  * In another words: only objects in parameters except array of options which 
 71:  * has to be in first place or can be absent.<br>
 72:  * 
 73:  * You have to call 
 74:  * <code>
 75:  *         parent::__construct($options);
 76:  * // or 
 77:  *         parent::__construct();
 78:  * </code> 
 79:  * in your constructor<br>
 80:  * 
 81:  * <b>About optional parametrs</b><br>
 82:  * All optional params will be ignored 
 83:  *(NULL will be retrived instead of them in to consructor).<br>
 84:  * 
 85:  * <br><b>What is Service Name?</b><br>
 86:  * It is just string - param for (@see Zaboy_Dic::get()}<br> 
 87:  * 
 88:  * <b>Object_Class::_defaultOptions</b><br>
 89:     * If Service Object is inherited from {@see Zaboy_Services} it contains
 90:     * {@see Zaboy_Services::_defaultOptions}<br>
 91:     * If $options haven't submitted via __construct - _defaultOptions will be used.<br>
 92:     * See {@see Zaboy_Services_Example_SimpleTest::testSetDefaultOptionsInConstruct()}<br>
 93:     * If $options is not empty - _defaultOptions will not be used fully ( any part)<br>
 94:     * See {@see Zaboy_Services_Example_OptionsTest::testSetDefaultOptionsInConstruct()}<br>
 95:  * 
 96:  * Also, you can define $options for Service in config.ini. 
 97:  * See about it (@see Zaboy_Dic_ServicesConfigs} <br>
 98:  * 
 99: 
100:  * @todo Object_Class :: _defaultOptions - rewrite docs
