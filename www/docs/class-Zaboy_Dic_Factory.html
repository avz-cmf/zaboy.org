Class Zaboy_Dic_Factory

This is factory for "make" service and objects which are created Zaboy_Dic

There is special method Zaboy_Dic_Factory::getOptionalParamValue() for lazy load. See Zaboy_Dic::getOptionalParamValue() and magic methods in Zaboy_Services.
